## 📖 About This Repository

**30 Days of Agentic AI** is a comprehensive, hands-on curriculum that takes you from understanding what an AI agent actually is, all the way to designing, building, evaluating, and deploying production-grade autonomous multi-agent systems.

This is not a theory dump. Every single day pairs a focused lesson with real code, a mini project, exercises, a quiz, and a cheat sheet you'll actually reuse on the job.

Inspired by the success of curricula like *30 Days of Python*, this repository applies the same "learn by building, one day at a time" philosophy to the fastest-moving field in software engineering: **agentic AI systems**.

### Who this is for

| Level | You should already know | You'll come out knowing |
|---|---|---|
| 🟢 Beginner | Basic Python (functions, loops, classes) | What agents are, how LLMs reason, how to call tools/functions |
| 🟡 Intermediate | General programming experience | RAG, memory systems, planning, ReAct, multi-step workflows |
| 🔴 Advanced | Software engineering background | Multi-agent orchestration, evaluation, guardrails, observability, deployment |

Suitable for students, software engineers transitioning into AI, ML engineers, founders building AI products, and researchers who want practical engineering skills.

---

## 🧠 Learning Philosophy

- **Learning by building** — every concept is paired with working code
- **Progressive difficulty** — Day 1 assumes nothing, Day 30 assumes everything before it
- **80% practical / 20% theory** — minimum viable theory, maximum hands-on practice
- **Daily hands-on projects** — not toy snippets, real mini-systems
- **Industry-focused workflows** — the same patterns used in production AI teams
- **Visual explanations** — Mermaid diagrams for every architecture
- **Reusable templates** — copy-paste starting points for your own agents
- **Production engineering mindset** — testing, evaluation, guardrails, and deployment are first-class citizens, not afterthoughts

## ⚙️ Getting Started

### Prerequisites
- Python 3.11+
- Git
- An API key for an LLM provider (OpenAI, Anthropic, or compatible) — set in `.env`
- (Optional) Docker, for Day 28+ deployment lessons

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/30-Days-Agentic-AI.git
cd 30-Days-Agentic-AI

# 2. Create a virtual environment
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate

# 3. Install base dependencies
pip install -r requirements.txt

# 4. Configure environment variables
cp .env.example .env
# then edit .env and add your API keys

# 5. Jump into Day 1
cd Day-01-Introduction-to-Agentic-AI
cat lesson.md
```

### Running a day's code

Each day's `code/` folder is self-contained:

```bash
cd Day-07-Tool-Using-Agents
pip install -r requirements.txt
python code/main.py
```

### Using Docker (optional, recommended for Days 21+)

```bash
docker-compose up --build
```
